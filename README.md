# Timer Queue 

Simplify working with consecutive timed tasks by adding them to a queue of timers. 

### Timers 

Timers are composable and re-usable for different tasks.

### Installation

````bash
$ npm install
````

### Development

````bash
$ npm start 
````

### Testing

````bash
$ grunt test
$ karma start
````

Visit [http://localhost:3000](http://localhost:3000)
